# Mine sweeper game
    Minesweeper clone made in C# WPF. As a week assignment ad EASV sønderborg DMU.

## Objectives 
    To win the game in the shortest amount of time

## GAME OVER
    1. A bomb is exposed.

## GAME WIN 
    1. All tiles have been reviled.
